# my_kemal_fbot
 
 This is a web app that consist several forums parsed by crawler

## Installation

1. cd fbot-kemal && shards install
2. [dev] crystal build src/main.cr
	 [release] crystal build src/main.cr --release


## Usage
to run web site
./main
and open web browser 
http://localhost:3000/
